# Career Lab

Career Lab is a work-in-progress SaaS application designed to support job seekers throughout the hiring lifecycle by combining resume optimization, application tracking, and interview preparation into a single platform.

The project focuses on practical, AI-assisted workflows that reflect real-world hiring and ATS evaluation processes.

---

## Core Capabilities

### Resume Analysis

Provides structured resume evaluation against a given job description, including:

- ATS compatibility scoring  
- Identification of missing and relevant keywords  
- Suggested experience statements aligned with role requirements  
- Recommended education phrasing  
- Skill gap analysis  
- Role-specific professional summary suggestions  

---

### Application Tracking

A centralized job application board that enables users to:

- Record job applications  
- Track application status and progress  
- Maintain visibility across multiple applications  

---

### AI Mock Interviews

Simulates real interview scenarios through an AI-driven interview experience:

- Voice-based interview interaction  
- Role and context-aware questioning  
- Post-interview feedback generated based on responses and performance
